# 📋 خطة تطوير تطبيق "بنّاء" - السيناريو المحدث

## 🎯 السيناريو الجديد

### المستخدم العادي:
1. تسجيل الدخول → الشاشة الرئيسية
2. عرض قائمة المشاريع والحسابات المخزّنة
3. إنشاء مشروع جديد (مساحة، طوابق، نوع خرسانة، تسليح)
4. حساب تلقائي للخرسانة والحديد والمواد
5. إدخال أسعار المواد الحالية
6. حساب التكلفة ومقارنة عروض الموردين
7. عرض أحدث العروض وفرزها
8. إنشاء تقرير (AR/EN/FR) وحفظ/مشاركة
9. إعدادات (إشعارات، وضع، ملف شخصي)

### الموردون:
1. تسجيل الدخول كـمورد
2. إدارة المتجر (موقع، مناطق التوصيل)
3. تحديد المواد المتاحة وأسعارها
4. تحديث الأسعار لحظياً
5. استلام إشعارات المشاريع الجديدة
6. إرسال عروض سعر خاصة
7. فترة صلاحية العرض

---

## 📦 الهيكلة المطلوبة

### قاعدة البيانات (Firestore):

```
users/
  {uid}/
    profile: {name, email, phone, photo, type}
    settings: {darkMode, notifications, language}
    projects/ (collection)
      {projectId}/
        name, type, floors, city
        concreteType, reinforcementType
        components: []
        materials: []
        totalCost
        createdAt
        status

suppliers/
  {uid}/
    profile: {name, shopName, phone, location, deliveryAreas}
    materials/ (collection)
      {materialId}/
        name, category, price, unit, lastUpdated
    offers/ (collection)
      {offerId}/
        projectId, userId
        price, validUntil, status
        materials: []

quoteRequests/
  {requestId}/
    projectId, userId
    status (pending/responded/accepted/rejected)
    responses: []
```

## 📱 الواجهات المطلوبة

### المستخدم العادي:
1. **شاشة登录** - Login/Register

3. **المشاريع** - Projects list
4. **مشروع جديد** - New Project wizard
5. **تفاصيل المشروع** - Project details
6. **الأبعاد** - Dimensions input
7. **الأسعار** - Material prices
8. **عروض الموردين** - Supplier offers
9. **التقرير** - Report generation
10. **الحاسبة** - Quick calculator
11. **الإعدادات** - Settings

### المورد:
1. **لوحة المورد** - Supplier dashboard
2. **إدارة المواد** - Materials management
3. **العروض المستلمة** - Received requests

## 🛠️ الوظائف الجديدة المطلوبة

### 1. نظام المشاريع المحسّن
- إضافة نوع التسليح (فيل®, شبك®, حلقي®)
- إضافة مراحل المشروع (أساس، سقف، ...)
- حساب تلقائي متعدد المراحل

### 2. نظام الأسعار
- قائمة أسعار المواد
- تحديث الأسعار من عدة موردين
- مقارنة تلقائية للأسعار

### 3. نظام عروض الموردين
- طلب عرض سعر
- استلام عروض متعددة
- مقارنة العروض
- قبول/رفض العرض

### 4. نظام التقارير المحسّن
- دعم اللغات (AR/EN/FR)
- تصدير PDF محسّن
- مشاركة متعددة
